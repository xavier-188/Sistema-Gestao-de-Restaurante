Sistema de Gestão para Restaurante

Sistema FullStack desenvolvido para gerenciamento de clientes e reservas de um restaurante.
O projeto foi criado com foco em aplicar conceitos de desenvolvimento backend, frontend e integração entre API e banco de dados.

Funcionalidades

### 👥 Clientes/Mesa
- Cadastro
- Listagem
- Busca por ID
- Atualização
- Exclusão

### 📅 Reservas
- Cadastro de reservas
- Controle de status
- Associação de informações do cliente
- Persistência em banco de dados

---

Tecnologias Utilizadas

### Backend
- C#
- ASP.NET Core
- Entity Framework Core
- API REST
- SQLite

### Frontend
- React
- TypeScript
- Axios
- React Router

---

Arquitetura

O projeto segue uma estrutura baseada em:
- Controllers/Endpoints para gerenciamento das rotas
- Entity Framework para persistência de dados
- Separação entre frontend e backend
- Comunicação via requisições HTTP (REST)

---

Como Executar o Projeto

### Backend
1. Clone o repositório
2. Acesse a pasta do backend
3. Execute: dotnet restore - dotnet run
  
### Frontend
1. Acesse a pasta do frontend
2. Execute: npm install - npm run dev
